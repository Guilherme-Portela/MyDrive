Vamos começar pelo início do roadmap, com os **Fundamentos da Programação**. Esses fundamentos são essenciais porque te dão uma base sólida para entender conceitos mais avançados no futuro.

### Passo 1: Lógica de Programação
1. **Conceitos Básicos**
   - **Variáveis**: Aprenda como armazenar dados em variáveis.
   - **Tipos de Dados**: Inteiros, floats, strings, booleans.
   - **Operadores**: Aritméticos (+, -, *, /) e lógicos (AND, OR, NOT).

2. **Estruturas de Controle**
   - **Condicionais**: if, else, elif.
   - **Loops**: for e while.

3. **Algoritmos Simples**
   - Sequência de passos para resolver um problema.
   - Exemplos: Algoritmo para somar números, encontrar o maior de três números, etc.

### Passo 2: Introdução ao Python
1. **Instalação do Python**
   - Baixe e instale o Python do site oficial: [python.org](https://www.python.org/).

2. **Primeiros Passos com Python**
   - Escreva e execute seu primeiro programa (ex: `print("Hello, World!")`).
   - Utilize um ambiente de desenvolvimento integrado (IDE) como o VS Code ou o PyCharm.

3. **Sintaxe Básica**
   - Declaração de variáveis.
   - Entrada e saída de dados (`input()` e `print()`).
   - Estruturas de controle (if, for, while).

4. **Estruturas de Dados**
   - Listas, tuplas, dicionários.
   - Métodos comuns para manipulação de listas (append, remove, sort).

5. **Funções**
   - Definição e chamada de funções.
   - Parâmetros e retorno de valores.
